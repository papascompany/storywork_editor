02프로젝트 비전

StoryWork 는 1,000+개의 PNG 포즈 라이브러리, 관리자 등록 리소스, AI 대본 분석 을 결합해 fabric.js 기반 캔바급 온라인 스토리보드 편집기 를 제공하고, 완성된 페이지를 POD(Print-On-Demand) PDF 로 출판한다.

대본만 넣으면 완성

로그인 없이 시작 → 대본 붙여넣기 → AI 가 장면 분할/포즈/배경 자동 배치 → 캔바 수준의 편집기로 후편집 → POD PDF 한 번에.

  • 1,260개 포즈 + 4 판형 프리셋 (B5/A5/정사각/세로형)
  • 장면 분할 F1 ≥ 0.8 (M4-01 목표)
  • 16페이지 PDF ≤ 6초
