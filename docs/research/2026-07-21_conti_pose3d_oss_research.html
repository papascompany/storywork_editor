2두 제안 요약

제안 A — 콘티 자동생성 모듈 CONTI-02 즉시 착수 가능

  • CONTI-01 장면 메타 실가동(LLM enhance)·SceneDoc.meta 영속화·Inngest 비동기 API 키
  • CONTI-02 콘티 시트 합성기(fabricJson)+conti-2x4 템플릿+PDF 키 불요
  • CONTI-03 콘티 미리보기 UI — 덮어쓰기 손실 리스크 구조적 회피 /ui-spec

시장 공백: 대본 자동분해 × 자산 조립 × 말풍선 × POD 인쇄의 교집합 제품 부재. 생성형(Boords·LTX·Katalist)은 재현성 없음, 네이버웹툰도 콘티 자동화는 공백. 포지션 = "완성작 생성기가 아닌 편집 가능한 콘티 에디터".

제안 B — 포즈 3D화 양산 베이스 4~10종만 3D화

  1. T/A-pose 턴어라운드 시트 → AI 이미지→3D (10종 $10~30) + 클린업 1패스
  2. 리깅(Rigify/AccuRig 무료·자사 소유) → 포즈 적용(사내 제작 주력)
  3. Blender headless: 툰셰이딩 + 4방위 카메라 → 2,048px+ PNG
  4. 본 관절 투영 → 정확한 25점 키포인트 사이드카 공짜
  5. 기존 ingest 그대로 재적재(수용측 개발 ≈ 0)

lowDpi 96%·키포인트 3점·뷰 커버리지 편차를 동시 해결. 재렌더=동일 출력(ADR-0007 정합). 크리티컬 패스는 기술이 아니라 합법적 포즈 소스(Mixamo 회색지대·AMASS 상업불가). 파일럿: 체형 1종×포즈 20×앵글 4 = $50 미만.
