# 인쇄소 프리셋 등록 절차

관리자가 새 인쇄소를 시스템에 추가하는 단계별 가이드.

## 1. admin /printers 페이지 접속

curator 이상 권한으로 로그인 후 좌측 메뉴 "인쇄소 프로필" 클릭.

## 2. "새 인쇄소" 버튼 클릭

상단 우측 녹색 버튼.

## 3. 폼 작성

### 기본 정보

- **인쇄소 이름**: 표시 이름 (예: "한국출판인쇄")
- **슬러그**: URL-safe 키 (예: `han-pub-print`) — 영소문자, 숫자, 하이픈만
- **설명**: 옵션. 사양 특이사항 기재

### 지원 판형

Format.name 기준 슬러그 목록. 쉼표 구분.
빈 값 = 모든 판형 허용.

예: `b5, a5, b5-format`

### Bleed / Safe 사양

| 항목 | 권장 |
|------|------|
| Bleed 최소 | 인쇄소 요구 최솟값 (보통 3mm) |
| Bleed 최대 | 허용 최댓값 (보통 5mm) |
| Safe Area 최소 | 내용 보호 여백 (보통 5mm) |

### 이미지 DPI

| 항목 | 기준 |
|------|------|
| 포즈/인물 최소 DPI | 인쇄 품질 기준 (보통 300) |
| 배경 최소 DPI | 배경은 낮아도 허용 (보통 150~200) |

### 폰트/색공간

- **폰트 임베드 필수**: 대부분 True
- **허용 색공간**: RGB만/CMYK만/모두 — 인쇄소 확인 필수

### 최대 페이지

0 또는 빈값 = 제한 없음.

### 커스텀 경고

인쇄 시 주의사항을 한 줄씩 입력. 사용자에게 preflight 결과에 표시됨.

## 4. 등록

"등록" 버튼 클릭 → 목록으로 이동.

## 5. 확인

등록 후 목록에서 활성(초록 점) 상태인지 확인.
비활성 상태라면 해당 행 클릭 → isActive 토글 ON.

## 참고

- isSystem=true 프리셋 (bookprint-korea/instaprint/comicmaker): 삭제 불가, isActive 토글만 가능
- 잘못 등록 시 superadmin 이 삭제 가능 (isSystem=false 만)
- 프리셋 변경은 즉시 preflight 검증에 반영 (DB 어댑터 우선)
